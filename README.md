# 🔗 URL Shortener

A Python/Flask URL Shortener with custom aliases, click tracking, and a beautiful dark UI.

## Live Demo :- https://url-shortener-production-2abb.up.railway.app

## ✨ Features
- Shorten long URLs instantly
- Custom aliases (e.g. snip.ly/my-link)
- Click tracking
- Search and delete links
- SQLite database
- Dark mode UI

## 🛠️ Tech Stack
- Python
- Flask
- SQLite
- HTML/CSS/JavaScript

## 🚀 Run Locally
```bash
pip install -r requirements.txt
python app.py
```
Open http://127.0.0.1:5000
